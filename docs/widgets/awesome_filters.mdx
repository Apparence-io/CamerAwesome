## Filters

CamerAwesome embeds a set of **filters** that can be applied to the camera preview.

There is actually **29** filters which can be applied on your pictures to make them look awesome ðŸŽ‰.

We are working hard to add more filters to the plugin but you can also [contribute](https://github.com/Apparence-io/CamerAwesome/pulls) by adding your own filters ðŸ˜Ž.

![Filters preview](/img/filters.gif)

You can add a filter directly in the camerAwesome builder:

```dart
CameraAwesomeBuilder.awesome(
  // [...]
  filter: AwesomeFilter.AddictiveRed,
),
```
